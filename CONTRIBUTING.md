## Development Setup

**No dependencies needed!** Veltix uses Python stdlib only.
```bash
git clone https://github.com/Nytrox/veltix.git
cd veltix

# That's it! Ready to code.
```

Run examples:
```bash
python examples/echo_server.py
```

Run tests:
```bash
python -m pytest  # If you want to run tests
# or just
python tests/test_basic.py
```

**Optional (for development tools only):**
```bash
pip install pytest black mypy  # Dev tools, not Veltix deps
```

## Code Style

- Follow PEP 8
- Use type hints
- Add docstrings to public methods
- Keep it simple and readable
