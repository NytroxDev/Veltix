# AI Use in Veltix

Veltix is a human-led project. I design, implement, and maintain the core
architecture myself. AI tools are used as productivity assistants for tasks
where they improve clarity, consistency, or development speed.

## Human-written and owned

- Core Veltix code (protocol, server, client, routing, handshake, etc.)
- Architecture and design decisions
- Protocol design
- API design
- Code review and validation
- Final approval of every change

## AI-assisted

AI tools may help with:

- Tests and test fixtures
- Documentation and technical writing
- Docstrings and type annotations
- Log messages and error wording
- Commit messages
- Minor refactoring and formatting
- Parts of tooling such as benchmarks or developer utilities

All AI-assisted contributions are reviewed, understood, tested, and approved
before being included in Veltix.
